# MARIO MONTEIRO PORTFOLIO
## React web app
This is a personal portfolio built with React to showcase my work as a Frontend Developer and UI/UX Designer. It features my projects and components, demonstrating dynamic and user-focused interfaces. In this portfolio you can learn more about my stacks, languages I can speak as well as have access to my most updated resume.

[See the app]([https://invoice-component.netlify.app](https://mariomonteiroportfolio.netlify.app))

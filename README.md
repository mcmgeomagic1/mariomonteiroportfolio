# MARIO MONTEIRO PORTFOLIO
## React web app
This is a personal portfolio built with React to showcase my work as a Frontend Developer and UI/UX Designer. It features my projects and components, demonstrating dynamic and user-focused interfaces. In this portfolio, you can learn more about my tech stack, languages I speak, and access my most up-to-date resume.

The project was initially designed in Figma, where I conducted a full UX process to better understand my own needs and deliver a clear, intuitive, and engaging experience for recruiters and visitors.

[See the app]([https://invoice-component.netlify.app](https://mariomonteiroportfolio.netlify.app))

![readme](readme.png)

### Technologies
* React.js
* React Router
* Context Api
* Framer Motion
* Styled-Components
* React Icons
* Formspree
* Horizontal Scroll
